# WorldVoice 用户指南

WorldVoice 是一款 NVDA 插件，支持 Espeak、OneCore 和 SAPI5 等不同语音引擎角色之间的自动切换，并提供各式各样的自定义设置。用户可依照不同情境及个人偏好，轻松调整语音设置。

随着全球化的普及，人们接触多国文字内容的机会也越来越多。无论是学习教材还是休闲阅读，我们经常可以看到不同语言混合使用，甚至在同一句话中出现中英日多种语言文字的情况。

各类数字内容（例如语言教材、数理信息或文学作品）在文字结构、上下文脉络和语言混用频率上都会有所不同。使用屏幕阅读器时，能够根据内容特性与个人喜好灵活调整朗读方式，可以提升阅读质量。

## 功能特性

* 可依不同语言指定专属语音角色，让多语内容朗读时自动切换到更合适的语音。
* 各语音角色可独立设置速度、音调、音量，在不同语言间保留各自的朗读设置。
* 可切换数字朗读方式，依需求选择以数值或逐字方式读出数字。
* 可调整数字、项目、中文与全文朗读间的停顿长度，使朗读节奏更符合使用习惯。
* 可根据 Unicode 字符自动检测语言，减少多语内容朗读时手动切换语音的需求。

## 语音角色

在 WorldVoice 的语音角色设置面板中，您可以为不同语言设置专属的语音角色，并针对每个角色分别调整速度、音调和音量等参数。

WorldVoice 会独立保存每个语音角色的设置值，切换角色时会应用相应的设置，让语音角色能拥有各自的速度、音调与音量，这项功能特别有助于阅读非母语的内容。

* 选择一个地区语言后，语音列表会显示该地区可用的语音角色。选择某个语音角色即可完成该地区与语音角色的对应关系。
* 选择语音角色后，如果该角色支持变声，将会显示可用的变声选项供用户选择。
* 选定语音角色后，速度、音调、音量滑块，以及加快语速的复选框将自动调整到该角色的设置值。调整设置值只会影响当前选定的语音角色（各个语音角色拥有各自独立的设置值）。

### 一致性设置：

* 保持主要语音引擎与地区语音引擎一致：启用后，主要语音角色与地区语音角色将只能是同个语音引擎下的语音角色。若调整主要语音角色时，发现地区语音角色原属不同语音引擎，则该地区的语音角色会恢复成未选择状态，并仅可重新选择与主要语音角色相同引擎下的语音角色。
* 保持主要语音角色与地区语音角色一致：启用后，主要语音角色与地区语音角色将保持一致。当您调整其中一方的语音角色设置时，另一方将自动同步为相同的语音角色。
* 保持不同语音角色之间的参数一致：启用后，各语音角色之间的速度、音调、音量及加快语速参数将保持一致。调整任何一个语音角色的设置值，所有其他语音角色的设置值都会同步更新。

主要语音角色是通过 NVDA 语音设置面板设置的，而地区语音角色则是通过 WorldVoice 的语音角色设置面板设置的。

## 语音流程

语音流程设置用来控制朗读时与数字、停顿及语言处理相关的行为。您可依需求决定这些设置要应用到所有支持的合成器，或仅在 WorldVoice 中使用。

* 支持全局的语音流程：忽略在数字间的逗号、数字模式、项目间停顿、数字间停顿、中文间停顿、读出全部间停顿
* 仅支持 WorldVoice 的语音流程：根据 Unicode 字符检测语言、数字语言

* 根据 Unicode 字符检测语言：启用后将根据文字 Unicode 自动判断语言地区并切换语音角色。注意：此功能可能与 NVDA 的自动语言切换功能冲突，建议避免同时启用。
* 数字语言：朗读数字时，使用此选项设置的地区角色进行朗读
* 数字模式：
  * 数值模式：将数字作为数值朗读，例如“12345”会读作“一万两千三百四十五”。
  * 数字模式：逐字朗读数字，例如“12345”会读作“一二三四五”。
* 语音停顿调整：设置数字之间、项目之间、中文之间与全文朗读之间的停顿时间，数值越小停顿越短，0 为不停顿。
* 忽略数字间的逗号：朗读数字时忽略数字间逗号，以提高特定的语音角色朗读数值的正确性。

支持全局的语音流程设置可通过 NVDA 的语音设置面板或 WorldVoice 的语音流程面板进行调整。而不支持全局的语音流程设置则仅能通过 NVDA 的语音设置面板进行调整。

## Unicode 检测

这些选项用来调整 WorldVoice 在检测文字语言时，如何处理数字、标点符号与多语共用字符集。

* 检测语言时忽略数字：勾选后，数字将忽略检测，并使用当前的语音角色进行朗读。
* 检测语言时忽略常见标点符号：勾选后，标点符号将忽略检测，并使用当前的语音角色进行朗读。
* 语言检测时间点：决定何时根据 Unicode 的语言检测和语言切换指令是在 NVDA 处理语音指令之前还是之后。
* 字符集假设的语言：为 unicode 中多重语言共用的字符集（包括拉丁字符、中日韩字符、阿拉伯字符）设置默认的语言。

## 注意

* 若您同时启用 NVDA 的自动语言切换与 WorldVoice 的 Unicode 检测，两者可能互相影响，建议依需求择一使用。
* 若您想停用特定的语音引擎，请在 WorldVoice 语音引擎面板中取消勾选相应的复选框。
* 若您需要使用 Espeak, OneCore, SAPI5 以外的语音引擎，请额外下载相应的驱动语音包。

# WorldVoice 更新日志

## WorldVoice v5.0 更新

*   **新增功能及设置面板：** 语音流程和日志记录
*   **移除已弃用的功能和设置面板：** Unicode 规范化、数字-点替换以及其他过时的设置项目
*   **新增语音流程功能：** 为所有合成器启用 WorldVoice 的语音流程
*   **新增日志记录功能：** 捕获每个流程前后的语音序列，以便于调试
*   **新增语速加倍功能：** 允许为特定语音引擎启用语速加倍
*   **增强用户体验：** 改进语音引擎和日志记录对话框中的交互体验
*   **修订界面文本：** 更新界面文本以统一风格

## WorldVoice v4.0 更新

*   将 Espeak 引擎集成到 WorldVoice 支持的引擎中。
*   在 NVDA 的语音设置面板中增加语速加倍设置项目。
*   在 NVDA 的语音设置面板中增加语音变声设置。
*   如果当前引擎或语音支持语速加倍，NVDA 的语音设置面板将动态显示语速加倍的设置界面。
*   用户可以在 WorldVoice 的语音设置面板中为单个语音设置语速加倍。
