# Руководство пользователя WorldVoice

WorldVoice — это дополнение для NVDA, которое поддерживает автоматическое переключение между речевыми ролями разных речевых движков, включая Espeak, OneCore и SAPI5, и предоставляет широкий набор настраиваемых параметров. Пользователи могут легко настраивать параметры речи для разных ситуаций и личных предпочтений.

По мере распространения глобализации люди всё чаще сталкиваются с многоязычным текстовым содержимым. Как в учебных материалах, так и при чтении для отдыха часто встречается смешение разных языков, а иногда китайский, английский и японский текст могут появляться даже в одном предложении.

Разные виды цифрового содержимого, например материалы для изучения языков, математическая информация или литературные произведения, отличаются структурой текста, контекстом и частотой смешения языков. При использовании программы экранного доступа возможность гибко настраивать способ чтения в соответствии с особенностями содержимого и личными предпочтениями может повысить качество чтения.

## Особенности

* Позволяет назначать отдельные речевые роли для разных языков, чтобы при чтении многоязычного содержимого автоматически выбирался более подходящий голос.
* Позволяет независимо настраивать скорость, высоту тона и громкость для каждой речевой роли, сохраняя отдельные параметры чтения для разных языков.
* Позволяет переключать режим чтения чисел и выбирать чтение чисел как значений или по цифрам.
* Позволяет настраивать длительность пауз между числами, элементами, китайским текстом и чтением всего текста, чтобы ритм чтения лучше соответствовал привычкам пользователя.
* Может автоматически определять язык по символам Unicode, уменьшая необходимость вручную переключать голоса при чтении многоязычного содержимого.

## Речевые роли

На панели настроек речевых ролей WorldVoice можно назначать отдельные речевые роли для разных языков и индивидуально настраивать для каждой роли скорость, высоту тона, громкость и другие параметры.

WorldVoice хранит настройки каждой речевой роли отдельно. При переключении роли применяются соответствующие настройки, позволяя каждой речевой роли иметь собственную скорость, высоту тона и громкость. Это особенно полезно при чтении содержимого на неродном языке.

* После выбора языка или региона список голосов показывает речевые роли, доступные для этого языка или региона. Выбор речевой роли завершает сопоставление языка или региона с речевой ролью.
* После выбора речевой роли, если она поддерживает варианты голоса, будут показаны доступные варианты для выбора.
* После выбора речевой роли ползунки скорости, высоты тона и громкости, а также флажок ускорения речи автоматически устанавливаются в значения этой роли. Изменение этих значений влияет только на текущую выбранную речевую роль, поскольку каждая речевая роль имеет собственные независимые настройки.

### Настройки согласованности:

* Сохранять согласованность основного речевого движка и региональных речевых движков: если включено, основная речевая роль и региональные речевые роли могут использовать только речевые роли одного и того же речевого движка. Если при изменении основной речевой роли региональная речевая роль относится к другому речевому движку, она сбрасывается в состояние без выбора, и её можно снова выбрать только из того же движка, что и основную речевую роль.
* Сохранять согласованность основной речевой роли и региональных речевых ролей: если включено, основная речевая роль и региональные речевые роли остаются одинаковыми. При изменении речевой роли с одной стороны другая сторона автоматически синхронизируется с той же речевой ролью.
* Сохранять согласованность параметров между разными речевыми ролями: если включено, скорость, высота тона, громкость и ускорение речи остаются одинаковыми для всех речевых ролей. Изменение настроек любой речевой роли синхронно обновляет настройки всех остальных речевых ролей.

Основная речевая роль настраивается через панель голосовых настроек NVDA, а региональные речевые роли настраиваются через панель речевых ролей WorldVoice.

## Речевой конвейер

Настройки речевого конвейера управляют поведением чтения, связанным с числами, паузами и обработкой языка. Вы можете выбрать, применяются ли эти настройки ко всем поддерживаемым синтезаторам или только в WorldVoice.

* Глобально поддерживаемые настройки речевого конвейера: игнорирование запятых между числами, режим чисел, пауза между элементами, пауза между числами, пауза между китайскими символами, пауза при чтении всего текста
* Настройки речевого конвейера только для WorldVoice: определение языка на основе символов Unicode, язык чисел

* Определять язык на основе символов Unicode: если включено, WorldVoice автоматически определяет язык или регион по символам Unicode в тексте и переключает речевые роли. Примечание: эта функция может конфликтовать с автоматическим переключением языков NVDA, поэтому рекомендуется не включать обе функции одновременно.
* Язык чисел: при чтении чисел используется региональная роль, выбранная этим параметром.
* Режим чисел:
  * Числовой режим: читает числа как значения. Например, "12345" читается как "двенадцать тысяч триста сорок пять".
  * Режим чтения по цифрам: читает числа по цифрам. Например, "12345" читается как "один два три четыре пять".
* Настройка речевых пауз: задаёт длительность пауз между числами, элементами, китайским текстом и чтением всего текста. Меньшие значения означают более короткие паузы, а 0 означает отсутствие паузы.
* Игнорировать запятые между числами: при чтении чисел игнорирует запятые между цифрами, чтобы повысить точность чтения числовых значений для определённых речевых ролей.

Глобально поддерживаемые настройки речевого конвейера можно изменять через панель речевых настроек NVDA или панель речевого конвейера WorldVoice. Настройки речевого конвейера, которые не поддерживаются глобально, можно изменять только через панель речевых настроек NVDA.

## Определение Unicode

Эти параметры настраивают, как WorldVoice обрабатывает числа, знаки препинания и наборы символов, общие для нескольких языков, при определении языка текста.

* Игнорировать числа при определении языка: если включено, числа игнорируются при определении языка и читаются текущей речевой ролью.
* Игнорировать распространённые знаки препинания при определении языка: если включено, знаки препинания игнорируются при определении языка и читаются текущей речевой ролью.
* Время определения языка: определяет, выполняются ли команды определения языка и переключения языка на основе Unicode до или после обработки речевых команд NVDA.
* Предполагаемый язык для наборов символов: задаёт язык по умолчанию для наборов символов Unicode, общих для нескольких языков, включая латинские символы, символы CJK и арабские символы.

## Примечания

* Если одновременно включить автоматическое переключение языков NVDA и определение Unicode в WorldVoice, эти функции могут влиять друг на друга. Выберите одну из них в соответствии со своими потребностями.
* Если вы хотите отключить определённый речевой движок, снимите соответствующий флажок на панели речевых движков WorldVoice.
* Если вам нужен речевой движок, отличный от Espeak, OneCore или SAPI5, отдельно загрузите соответствующий драйвер или голосовой пакет.

# Журнал обновлений WorldVoice

## Обновление WorldVoice версии 5.0

* **Добавлены функции и панели настроек:** Речевой конвейер и запись журнала
* **Удалены устаревшие функции и панели настроек:** нормализация Unicode, замена цифр и точек, а также другие устаревшие настройки
* **Добавлена функция речевого конвейера:** включение речевого конвейера WorldVoice для всех синтезаторов
* **Добавлена функция записи журнала:** запись речевых последовательностей до и после каждого шага конвейера для упрощения отладки
* **Добавлена функция ускорения речи:** возможность включать ускорение речи для определённых речевых движков
* **Улучшен пользовательский опыт:** улучшено взаимодействие в диалогах речевых движков и записи журнала
* **Обновлён текст интерфейса:** текст интерфейса обновлён для единого стиля

## Обновление WorldVoice версии 4.0

* Движок Espeak интегрирован в список движков, поддерживаемых WorldVoice.
* В панель голосовых настроек NVDA добавлена настройка ускорения речи.
* В панель голосовых настроек NVDA добавлена настройка варианта голоса.
* Если текущий движок или голос поддерживает ускорение речи, панель голосовых настроек NVDA динамически отображает интерфейс настройки ускорения речи.
* Пользователи могут настраивать ускорение речи для отдельных голосов на панели голосовых настроек WorldVoice.
